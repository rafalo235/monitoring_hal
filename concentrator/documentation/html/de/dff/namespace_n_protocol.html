Detailed Description

NProtocol jest przestrzenia nazw klas odpowiedzialnych za wysylanie danych.

Plik naglowkowy protokolu komunikacji pomiedzy serwerem a koncentratorem systemu monitorowania hal.

protocol.h

Author
: Marcin Serwach
Version
1.0
Date
4.07.2014
 C++ standard: C++11

 Zastosowano przedrostki:
 E - enum
 N - namespace
 S - struct
 C - class

 Formatowanie: doxygen

 Zalozenia:
 1) sizeof(float) = 4
 2) sizeof(double) = 8
 3) sizeof(long long) = 8
 4) Kolejnosc bajtow Little Endian (ewentualna konwersja powinna zostac przeprowadzona przed zapisem)
